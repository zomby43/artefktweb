Quickstart

The shortest path from source image to exported file.

Open a source image

Use File > Open Image, the empty canvas button, or drag an image onto the canvas. Artefkt accepts PNG, JPEG, WebP, HEIC and HEIF source images.

Add effects

Search or browse the Effect Library. Double-click an effect, or press its add button, to place it into the stack.

Build the stack

Effects run from top to bottom. Reorder layers, toggle them, duplicate them, adjust opacity, and choose blend modes per layer.

Tune and compare

Select a stack layer to edit its parameters. Use seeds, presets, advanced controls and before/after view while the preview updates.

Export

Use Export for a save dialog, or Quick Export after setting a destination folder. Export supports PNG, JPEG and WebP.

Workspace

The app is built around a source image, a preview canvas and an editable stack.

Top bar
Project actions, view controls, undo/redo, settings, help, Quick Export and Export.
Effect Library
Searchable effect list with categories, favorites, recents and add controls.
Canvas
Live preview with zoom, pan, fit, actual size and before/after comparison.
Effect Stack
Ordered layer stack with visibility, opacity, blend mode, duplicate, remove and preset controls.
Params pane
Controls for the selected effect. It can live in the right pane or in the optional bottom Device Strip.
Status bar
Preview size, render timing, zoom, export mode and progress feedback.
Auto-Reveal Focus Mode
Hides enabled side panels and the Device Strip while leaving the canvas visible. Hover the left, right or bottom edge to temporarily reveal its panel.

Focus Canvas vs Auto-Reveal Focus Mode

Focus Canvas hides panels until you restore them manually. Auto-Reveal Focus Mode keeps the canvas clear while temporarily revealing enabled panels when your pointer reaches an edge; leave the panel and it hides again.

Preview and export

Preview is optimized for editing speed. Export is where the final render happens.

Preview is not the source file

By default, Artefkt builds a downscaled preview so parameter changes stay responsive. The app can still export from the source image size by rerunning the same stack during export.

Preview mode

Preview uses a configurable long-edge cap, resampling mode and optional source-resolution mode. Source-resolution preview is useful for checking detail, but it is heavier.

Export mode

Export can use source resolution for full-size output, or preview resolution when you want the current preview dimensions. The status bar shows the active export mode.

Files

Artefkt keeps source files, projects, exports and app data separate.

Source images
PNG, JPEG, WebP, HEIC and HEIF. Up to 100 MB, 48 MP, or 12000 px on either side.
Exports
PNG, JPEG and WebP. JPEG quality is controlled in Settings.
Projects
.glitch files save the stack, export options and source image path. They do not embed the source image.
User data
.artefkt files can export/import custom themes, stack presets, effect presets, app settings and library data.
Palettes
.hex, .png and .ase palette files. Imported palettes are deduplicated and capped at 256 colors.

Project files are recipes

A .glitch project stores the stack and the source image path. It does not include the original image bytes, so keep the source image beside your project or relink it if it moves.

Storage and locations

Where the Windows application is installed, what it stores automatically, and what remains under your control.

Installed application
The Windows installer currently places Artefkt at %LOCALAPPDATA%\Artefkt\, including artefkt.exe and uninstall.exe.
Application storage
%LOCALAPPDATA%\net.zombot.artefkt\EBWebView\ contains WebView-managed settings, themes, presets, library preferences and runtime/cache data.
Projects and exports
Source images, exported artwork and .glitch project files stay in the locations you select when opening or saving files.
Quick Export output
Quick Export writes to the destination folder configured in Settings.
Backups and transfers
.artefkt user-data backups are written wherever you choose during export.

Back up before resetting

Export an .artefkt user-data file before clearing application storage if you want to keep settings, themes or presets. Keep each .glitch project with its original source image because projects reference an image path rather than embedding the image.

Uninstalling and local data

Uninstalling removes the installed application. Your saved images, exports, projects and backups remain in the locations you chose. For a complete reset, remaining WebView application storage may need to be removed separately.

Effects and presets

The current app ships with 65 effects across six categories.

Color
Brightness, contrast, curves-style passes, channel tools and other color operations.
Palette
Palette map, palette cycle, indexed color drift and heatmap-style mapping.
Dither
Bayer, Atkinson, blue-noise, ordered pattern and error-diffusion style dithers.

Pixelation, resize, mosaic, fisheye, slice and related pixel transforms.

CRT, ASCII, hardware limits, halftone, scanlines and display-system treatments.

Stack presets

Save a whole effect stack as a reusable look. Stack presets can be applied later and can also be shared as stack strings.

Effect presets

Save settings for a single effect. Effects can also include built-in presets, seed controls and advanced parameters.

Settings

Settings are saved on the device unless you export them as user data.

Quick Export
Choose a folder, filename pattern, overwrite behavior and output format. Filename patterns support {name} and {timestamp}.
Preview
Adjust preview cap, source-resolution preview and resampling. Lower preview settings keep heavy stacks responsive.
Export resolution
Use source resolution for full-size output, or preview resolution when you want the current preview size.
Layout
Choose right-pane or bottom-strip parameters, insertion behavior, visible panels, Focus Canvas and Auto-Reveal Focus Mode.
Appearance
Use built-in themes, create custom themes, adjust GUI scale, choose UI fonts and control checker opacity.
Accessibility
Reduced motion, boosted secondary text, high contrast options and category color modes.
Data
Export or import local settings, themes and presets as an .artefkt file.

Requirements

The current release is built for Windows desktops.

System

Windows 10/11, 64-bit. Source images can be PNG, JPEG, WebP, HEIC or HEIF, and must fit the app limits: 100 MB, 48 MP, and 12000 px on either side.

Runtime

Artefkt runs locally after installation. Editing and exporting do not require an account check or a constant internet connection.

Installer security notice

Artefkt is currently distributed without a Windows code-signing certificate, so Windows SmartScreen may show an Unknown publisher warning when you launch the installer. Download the installer only from the official purchase/download page. To proceed through SmartScreen, select More info and then Run anyway.

The download includes a SHA-256 checksum. In PowerShell, run Get-FileHash ".\Artefkt-Setup-1.0.0-x64.exe" -Algorithm SHA256 and compare the result with the supplied checksum before installing.

Troubleshooting

Most problems come from moved source files, heavy preview settings or unsupported file formats.

A project opens without its image
The .glitch file stores the source image path, not the image bytes. If the image moved or was deleted, choose a PNG, JPEG, WebP, HEIC or HEIF replacement image when prompted.
A drag-dropped image does not relink later
Some drag/drop paths are unavailable to the app. Save the original image somewhere stable and relink it if needed.
Preview feels slow
Lower the preview cap, turn off source-resolution preview, hide effect thumbnails, or reduce very heavy stacks while editing.
Panels flicker or repaint incorrectly
Open Settings > Accessibility and enable Safe Graphics Mode. It reduces WebView rendering effects for Windows scaling, WebView2, or GPU-related UI glitches.
Export differs from the preview size
By default, export can rerun the stack on the source image. Switch export resolution to preview if you want preview-sized output.
Quick Export opens the normal dialog
Quick Export falls back until a destination folder is configured in Settings.
Palette import fails
Check that the file is .hex, .png or .ase. Palette PNG files have their own size limit, and ASE import reads RGB color blocks.
I want to reset Artefkt completely
Export user data first if needed, uninstall Artefkt, then remove remaining application storage under %LOCALAPPDATA%\net.zombot.artefkt\ only when you intend to start clean. Reinstall afterward.

FAQ

Short answers for purchase, local files and sharing.

Is Artefkt a subscription?

No. The current plan is a one-time $5 purchase.

Does it need an account?

No account check is part of the local editing workflow.

Does it need the internet to edit?

No. Editing, preview and export work locally after installation.

Does it upload my images?

Artefkt works on local files. Images and projects stay local unless you export, import or share them yourself.

Does a project include my source image?

No. A .glitch project stores the stack and the source image path. Keep the original image if you want the project to reopen with it.

Can I share looks?

Yes. Use stack presets, shared stack strings, effect presets, custom themes or .artefkt data export depending on what you want to move.

Can people copy the app?

Artefkt is sold without DRM. Buying it supports future updates, but the app is not built around account locks and licenses.

Is Mac or Linux supported?

Not in the current Windows release, BUT IT IS A PRIORITY! :)
